bubble sort

Compare adjacent elements and swap until the array is sorted.


for (let i = 0; i < n - 1; i++) {
  for (let j = 0; j < n - 1 - i; j++) {
    if (arr[j] > arr[j + 1]) {
      [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
    }  }  }
How it works

Two nested loops: the outer loop runs over passes, the inner one over adjacent pairs. On each pass we compare elements pairwise and swap when the left is greater than the right. The largest value in the unsorted part bubbles to the end, so each pass fixes one more element at the tail.
